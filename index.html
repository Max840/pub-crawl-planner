How It Works

The Pub Crawl Planner makes it super easy to organize pub crawl events. You give it a bunch of pub locations and it will generate a crawling schedule for every team making sure they walk the shortest path between pubs. The Pub Crawl Planner will make sure that two teams will never visit the same pub at the same time.
